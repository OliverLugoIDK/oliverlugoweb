# Oliver Lugo Studio — Sitio web editable

Sitio estático profesional para diseñador gráfico: landing principal + páginas internas + servicios + paquetes + portafolio + contacto.

## Cómo verlo

Opción rápida: abre `index.html` en tu navegador.

Opción recomendada:

```bash
cd oliver-lugo-designer-site
python -m http.server 8000
```

Luego entra a `http://localhost:8000`.

## Estructura

- `index.html`: página principal.
- `servicios.html`: servicios detallados.
- `paquetes.html`: paquetes/precios.
- `portafolio.html`: galería de trabajos.
- `sobre-mi.html`: perfil profesional.
- `contacto.html`: formulario que abre WhatsApp.
- `gracias.html`: página de confirmación para integraciones futuras.
- `guia-edicion.html`: guía rápida de modificación.
- `assets/js/config.js`: datos editables de marca, servicios, paquetes, contacto, FAQ, etc.
- `assets/css/styles.css`: diseño, responsive y animaciones.
- `assets/img/`: logos, mockups y placeholders.

## Edición rápida

1. Abre `assets/js/config.js`.
2. Cambia nombre, WhatsApp, email, redes, servicios, paquetes, textos e imágenes.
3. Sustituye imágenes en `assets/img/`.
4. Publica toda la carpeta completa.

## Importante

Este sitio no copia código, imágenes ni textos privados de terceros. Es una versión original con estructura comercial similar: hero, prueba visual, servicios, paquetes, bonos, autoridad, FAQ y contacto.
